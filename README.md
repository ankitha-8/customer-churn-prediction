# 📊 Customer Churn Prediction

## 📌 Overview
Customer churn refers to when customers stop using a company's service.  
This project builds a machine learning model to predict whether a customer will churn or not, helping businesses take proactive steps to retain customers.

---

## 🎯 Problem Statement
The goal of this project is to analyze customer data and predict churn using machine learning algorithms.  
By identifying high-risk customers, companies can improve retention strategies and reduce revenue loss.

---

## 📂 Dataset
- **Dataset**: Telecom Customer Churn Dataset  
- **Features include**:
  - Tenure
  - Monthly Charges
  - Total Charges
  - Contract Type
  - Payment Method
  - Internet Service
- **Target Variable**:
  - Churn (Yes/No)

---

## ⚙️ Workflow

1. **Data Cleaning**
   - Handled missing values
   - Converted data types

2. **Exploratory Data Analysis (EDA)**
   - Analyzed churn distribution
   - Visualized relationships between features

3. **Feature Engineering**
   - Encoded categorical variables
   - Selected important features

4. **Model Building**
   - Trained multiple ML models

5. **Model Evaluation**
   - Compared models using accuracy and other metrics

---

## 🤖 Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest

---

## 📈 Results
- **Best Model**: Random Forest *(or update based on your result)*  

---

## 🔍 Key Insights
- Customers with **higher monthly charges** are more likely to churn  
- Customers with **long-term contracts** are less likely to churn  
- **New customers** have a higher churn rate  
- Certain payment methods are associated with higher churn
